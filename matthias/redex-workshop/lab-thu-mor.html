13 Lab Abstracting Abstract Machines I

Goals

— develop the world’s smallest static analysis

— define approximation on stores

— import semantic ideas into analysis

Start with and edit tutorial.rkt to complete the exercises.

Exercises

Exercise 12. Design the approximation relation for stores, named ⊑Σ. (Marked with FIXME in tutorial.rkt.)

Exercise 13. Add set! and void to the model starting with –>vσ and carry it through to –>vσ^. Can you design a more precise, yet sound, way of approximating store updates?

Exercise 14. Implement a garbage collector for –>vσ and carry it through to –>vσ^. What do you notice about the precision of the resulting analyzer?
