Early Exercise with Images: Making Fabric and Clothing

© Kathi Fisler, 2004

This exercise has students create images of fabrics, create clothing items from fabrics, and put logos onto clothing items. It is designed to give students practice composing functions.

Prerequisites: Numbers, Functions, Images, Defining Constants (Section 3.2)

Note to Teachers from Providence 2004 Workshop: this version differs slightly from the one we used in the workshop:

  • the basic shirt image is on a tan background (avoids confusion that the seemingly white background from the original image wasn't really white)
  • there's now a hat image
  • the backgrounds on the flower, worm, and chili images have be cleaned up so that there is no longer a white cloud around them.
  • In the teachpack for version 208, dye-fabric is currently disabled (we're still migrating that to 208) -- you can create similar fabrics without it by starting with colored fabrics rather than dying fabrics later.
