Exercise 2.2.5.  

putting it all together
Modify the function move from exercise ex:move so that the UFO reacts to keyboard events. Use process-command from exercise ex:keyevent to process events from the keyboard.

Use these functions to get started: 

;; String  -->  String
(define (main name)
  (go 
   (drop-dot (make-posn (/ WIDTH 2) 0) (ran-interval WIDTH WIDTH))
   name))

;; Boolean String  -->  String
(define (go result name)
  (cond
    [result (string-append name " is a winner")]
    [else   (string-append name " is a loser")]))
...
(go "your name here")
